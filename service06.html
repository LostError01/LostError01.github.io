Proyectos de Programación Creativa

Resumen

Creación de proyectos de programación creativa con Java/Javascript, C, C++, C# o HTML/CSS con motivos estéticos y/o experimentales ya sea mediante el uso de software o su complementación con dispositivos externos.

La creatividad llevada al código

Ofrezco servicios de creación artística a través de la programación creativa, utilizando una variedad de lenguajes, enfocados en el desarrollo de instalaciones interactivas, algoritmos generativos y sistemas multimedia. Mi trabajo explora la posibilidad expresiva del código, aplicándolo a proyectos que fusionan arte, tecnología y experiencia sensorial, con un enfoque en la experimentación y la exploración de nuevas formas de interacción.Con especialidad en lenguajes como:

  • Java mediante la interfaz de Processing
  • HTML, CSS y Javascript para entornos web.
  • C,C++ y C# para proyectos enfocados a trabajar en motores gráficos

Además, ahora mismo estoy cursando y aprendiendo el lenguaje de Kotlin para implementar este tipo de proyectos en dispositivos móviles.
